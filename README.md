### How is it build?
In this project, I used:
  - HTML
  - CSS
  - JavaScript
  - jQuery
  - Ajax

In addition, I used some other libraries:

* [AOS Animation](https://michalsnik.github.io/aos/) - Awesome animation on scroll library
* [Boostrap](https://github.com/twbs/bootstrap) - I guess you what it is
* [Font Awesome](https://github.com/FortAwesome/Font-Awesome) - Icons library
* [Hamburger Menu](https://github.com/jonsuh/hamburgers) - Great library of animated burger icons
* [popper.js](https://github.com/popperjs/popper-core) - I used it for Bootstrap
